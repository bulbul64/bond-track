# 🚀 BondTrack

### The People Who Matter, All in One Place

BondTrack is a modern relationship management web app that helps you stay connected with the people who matter most. Track interactions, monitor relationship health, and never miss important moments again.

🔗 **Live Site:** https://bond-track-alpha.vercel.app/

---

## 📌 About the Project

BondTrack is designed to help you organize and maintain meaningful relationships. Whether it's friends, mentors, or colleagues — you can track your interactions and get insights on who needs your attention.

---

## 🛠️ Technologies Used

* **Next.js (App Router)**
* **React.js**
* **TypeScript**
* **Tailwind CSS**
* **shadcn/ui**
* **Lucide Icons**
* **Vercel (Deployment)**

---

## ✨ Key Features

### 1. 📊 Smart Relationship Dashboard

* View total connections, strong bonds, and people who need attention
* Track monthly interactions (calls, messages, meetups)

### 2. 👥 Connection Management

* Add and manage people in your circle
* Tag relationships (e.g., mentor, team, startup)

### 3. ⏱️ Interaction Tracking

* See last interaction time
* Get status like:

  * ✅ On Track
  * ⚠️ Almost Due
  * ❌ Overdue

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

```bash
# 1. Clone the repository
git clone https://github.com/your-username/bondtrack.git

# 2. Go to project folder
cd bondtrack

# 3. Install dependencies
npm install

# 4. Run the development server
npm run dev
```

👉 তারপর ব্রাউজারে ওপেন করো:
`http://localhost:3000`
